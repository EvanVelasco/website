Not bad! But let's see if we can do even better. We can tune our model by finding the optimal k. To do this, we will create multiple KNN models using a variety of different values for N. We can then compare accuracy scores to determine the best model.

In [15]:
test = []
for i in range(0,9):
    knn = KNeighborsClassifier(n_neighbors=i+1, metric='euclidean')
    knn.fit(X_train, y_train)
    score = knn.score(X_test,y_test)
    test.append(score)
In [16]:
score_df = pd.DataFrame({'n_neighbors': [1,2,3,4,5,6,7,8,9], 'accuracy': test})

accuracy_plot = sns.barplot(data=score_df, x='n_neighbors',y='accuracy').set(title='KNN Accuracy by N_Neighbors')

The default for the KNN algorithm is k=5. From the data above we can see that this is not the optimal value. Instead, we will choose k=3.

Let's see how our model's score improves using this new k.

In [17]:
